== Description ==
RealAds is a plugin that helps you insert ads in your posts and pages.
To get un account, please contact us at [ads@hadran.net](mailto:ads@hadran.net).

== Installation ==
1. Upload the Plugin files to the `/wp-content/plugins/lap-api` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the Plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->RealAds settings screen to configure the plugin and enable the API.

== Frequently Asked Questions ==
= How do I get an account? =
Please contact us at [ads@hadran.net](mailto:ads@hadran.net).
= Why i can't see any ads? =
You must enable the Plugin in the settings page.
= How can i add ads to my site? =
It depends on the type of ads you want to display.
For more information, read the [documentation](https://lap-api.com/admin/docs).
